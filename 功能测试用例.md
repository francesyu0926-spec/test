# 招投标检测平台快控项目列表页功能测试用例

## 1. 截图内容分析

截图展示的是“招投标检测平台”的“快控项目”列表页，当前顶部页签为“AI审计试用”。页面主要结构如下：

- 顶部区域：平台 Logo、顶部页签“AI审计试用 / 数据展示应用 / 智慧审计大屏”、右上角用户账号和退出入口。
- 左侧菜单：包含“我的账户、我的项目、快控项目”，当前“快控项目”蓝色高亮；左侧还有收起/展开入口。
- 查询区域：包含项目名称输入框、招标方输入框、开标时间日期选择框、“查询”和“重置”按钮。
- 操作区域：包含“新建项目”主按钮和“刷新项目”按钮。
- 项目列表：表格列包含序号、项目名称、标段、开标时间、招标方、校验任务状态、操作。
- 状态与操作：校验任务状态展示“已完成/剩余”数量；项目名称为蓝色可点击链接；操作包含“编辑”“立即执行”“重新执行”，部分操作置灰不可点击。
- 数据统计：页面右下角展示“共12条数据”。

## 2. 测试范围

- 页面加载、顶部页签、左侧菜单、用户区域和布局展示。
- 项目名称、招标方、开标时间筛选查询，以及重置、空结果、异常处理。
- 新建项目、刷新项目、编辑、立即执行、重新执行、项目名称详情入口。
- 项目列表字段、状态文案、操作可用/置灰、总数展示。
- 权限控制、接口异常、并发点击、空数据、大数据量、兼容性和安全性。

| 编号 | 前置条件 |

| P-03 | 系统存在可编辑、可立即执行、可重新执行、无操作权限或任务执行中等不同状态项目。 |

| P-05 | 项目列表、新建项目、刷新项目、编辑、执行任务等接口可访问。 |
