# 产品生产追溯页面功能测试用例

## 1. 测试范围

基于截图页面整理，覆盖产品生产追溯/工况工艺详情页的主要功能：

- 页面进入、菜单导航、标题与模块展示
- 顶部基本信息/文件列表展示与附件查看
- 各业务检查表格的数据展示、状态标识、查看入口
- 工艺路线/过程参数矩阵的定位、选中、高亮、告警展示
- 人员资质、文件资料、关键尺寸/检测数据等明细表格
- 空数据、异常数据、权限、兼容性与易用性

## 2. 截图识别的页面元素

| 区域 | 截图可见元素 | 主要验证点 |
| --- | --- | --- |
| 顶部区域 | 系统 Logo、顶部导航、当前用户信息、页面标题 | 导航选中、标题准确、用户信息展示、页面刷新可恢复。 |
| 左侧菜单 | 左侧蓝色选中菜单、菜单图标 | 当前菜单高亮、点击切换、菜单收起/展开后页面正常。 |
| 顶部文件表格 | 文件名称蓝色链接、数量/状态等字段、“查看”操作 | 文件列表展示、链接打开、查看入口、无权限提示。 |
| 检查结果表格 | 序号、检查项、结果状态、操作列 | 正常/异常状态颜色、查看详情、空数据展示。 |
| 矩阵表格 | 横向工序列、纵向项目行、蓝色选中单元格、百分比/数值 | 单元格数据准确、点击高亮、横向滚动、行列对齐。 |
| 人员信息表格 | 人员编号/姓名、岗位或资质字段 | 人员资质展示、缺失或过期告警、多人员展示。 |
| 文件/资料异常模块 | 红色异常标识、红色告警三角图标、说明文字 | 模块级异常、行级异常、异常说明查看。 |
| 数值检测表格 | 标准值、测量值、偏差/比例、合格相关字段 | 数值格式、边界值、越界告警、单位展示。 |
| 底部说明区域 | 蓝色提示/说明条 | 说明内容展示完整，不遮挡或错位。 |
| 右侧目录 | 模块锚点导航、当前模块高亮 | 点击定位、滚动联动、高亮同步。 |

## 3. 测试前置条件

| 编号 | 前置条件 |
| --- | --- |
| P-01 | 测试账号可登录系统，并拥有查看生产追溯详情页权限。 |
| P-02 | 系统存在至少 1 条完整生产追溯记录，包含图纸/文件、工艺路线、人员、参数、检验数据。 |
| P-03 | 系统存在至少 1 条包含异常/告警状态的数据，用于验证红色告警图标与异常状态。 |
| P-04 | 系统存在至少 1 条部分模块无数据的记录，用于验证空状态展示。 |
| P-05 | 浏览器网络正常，文件预览/下载服务可访问。 |

## 5. 回归测试重点

| 回归点 | 验证内容 |
| --- | --- |
| 数据准确性 | 页面各模块字段值与接口/数据库测试数据一致。 |
| 告警一致性 | 模块级、行级、单元格级异常提示同时准确。 |
| 表格交互 | 查看、点击、高亮、滚动、空状态均可用。 |
| 文件能力 | 文件名称链接和“查看”入口可正确预览/下载并处理异常。 |
| 权限控制 | 页面权限、文件权限、非法 URL 均能正确拦截。 |

## 6. 建议测试数据

| 数据集 | 数据特征 |
| --- | --- |
| D-01 完整正常数据 | 所有模块均有数据，全部状态正常/通过。 |
| D-02 单模块异常数据 | 仅一个模块存在异常，用于验证模块告警定位。 |
| D-03 多模块异常数据 | 文件、人员、矩阵、检测数据同时存在异常。 |
| D-04 空数据 | 附件、检查项、人员、矩阵部分为空。 |
| D-05 大数据量 | 文件超过 20 条，矩阵列数超过可视范围，表格行数超过 100 条。 |
| D-06 权限受限数据 | 当前用户可看页面但不可查看部分文件或敏感字段。 |
